
# Digital Logic Design using Logisim

A collection of digital circuit designs created using Logisim.

## Projects Included

- 8-bit ALU
- Basic control unit
- Register File
- 2×2 Array Multiplier
- Basic Processor
- Memory operations (read/write)

## Software Used

- Logisim Evolution

## Concepts Covered

- Combinational Logic
- Sequential Logic
- Arithmetic Circuits
- Register Design
- Data Path Components

## Repository Structure

Each project contains:

- Logisim circuit file (.circ)
- Circuit screenshot
- Project description
